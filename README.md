TP3 : Gestion d'un Parking en Java
​Description
​Ce projet est une application Java réalisée dans le cadre du TP3. Il permet de gérer un système de parking en utilisant les concepts fondamentaux de la Programmation Orientée Objet (POO).
​Concepts Utilisés
​Héritage : Utilisation d'une classe mère Vehicule et de classes filles Voiture, Moto, et Camion.
​Polymorphisme : Manipulation de différents types de véhicules à travers un tableau de type Vehicule.
​Encapsulation : Utilisation d'attributs privés avec des constructeurs et des méthodes spécifiques.
​Structure du Projet
​Le projet contient les classes suivantes :
​Vehicule.java : Classe de base contenant les propriétés communes (Marque, Modèle).
​Voiture.java : Classe héritant de Vehicule avec des spécificités.
​Moto.java : Classe héritant de Vehicule.
​Camion.java : Classe héritant de Vehicule.
​TestParking.java : La classe principale (Main) qui contient le tableau et affiche les détails de chaque véhicule.
